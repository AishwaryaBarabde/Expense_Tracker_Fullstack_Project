# Expense Tracker Application

## Tech Stack
- Frontend: React
- Backend: Flask (REST API)
- Database: PostgreSQL
- Charts: Matplotlib

## Features
- Add Expense
- Edit Expense
- Delete Expense
- Monthly Summary
- Category-wise chart

## Setup Instructions

### Backend
cd backend
pip install -r requirements.txt
python app.py

### Frontend
cd frontend
npm install
npm start
